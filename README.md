# ResearchFin 💰
### Student Research Finance Advisor | MUJ DSE3170 PBL Project

A full-stack web application that helps B.Tech students manage and overcome
financial constraints during research projects.

---

## 🚀 Features
- 💸 **Budget Planner** — track expenses and calculate savings from free tools
- 🏦 **Funding Finder** — 25+ real sources (AICTE, DST, UGC, SIH) with apply guides
- 🛠 **Free Tools Directory** — 22 open-source alternatives to MATLAB, SolidWorks, ANSYS
- 📊 **Financial Risk Assessment** — 8-question quiz with personalised recommendations
- 📁 **Case Studies** — 5 real projects that cut costs by 55–100%
- 💡 **Strategy Framework** — 6 actionable cost-reduction strategies
- 🔐 **Authentication** — JWT login, bcrypt passwords, role-based access

---

## 📊 Key Results
- 78% average cost reduction across 5 real case studies
- ₹21,500 saved in one project by replacing just 4 tools
- 100% cost achieved in 2 case studies using only free tools
- 60–70% of students face financial stress (AISHE + NSSO data)

## ⚙️ How to Run

**Backend:**
```bash
cd Backend
npm install
node server.js
```

**Frontend:**
Open `Frontend/Research_Fin.html` in your browser
or use VS Code Live Server.
